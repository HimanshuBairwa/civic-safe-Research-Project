## 1. Remaining real issues (prioritized, with file:line)

### [A] 🟠 Docs overclaim standard methods as "Novel Contribution"
`MATHEMATICS.md` tags as novel: CRPS-direct training (§6), Adaptive Temporal ECRC (§8 — this **is** Gibbs–Candès 2021 ACI applied per-group), r-collapse reg (§9), SAC loss (§10 — the harmful term), EMOS-for-ZINB (§11.4). A reviewer who knows the literature reads these tags as inexperience and hunts for reasons to reject.
- **Fix:** demote each to "Applied method (cite X)". Real novelty is the Feedback Law (§3), not these. Cite **STZINB-GNN (Zhuang et al., KDD 2022)** and **STMGNN-ZINB (2024)** as the prior art the ZINB/GNN engine builds on — their absence from Related Work is currently the single biggest rejection risk.

### [B] 🟠 ASC ("Anomaly Skill Coefficient") interpretation is still inverted
`feedback_loop.py:81-87`: ">0 ⇒ predictions amplify the historical trend (risk)". But a *perfectly accurate* model correlates with true deviations → ASC≈1 → falsely branded "amplifier". ASC is the anomaly-correlation **skill** score, not a feedback measure.
- **Fix:** relabel ASC honestly as a skill diagnostic; route all feedback/fairness claims through `DownstreamAllocationDisparity` (already present and correct) and the new closed-loop `theory/` module.

### [C] 🟠 `scripts/aobf_benchmark.py` is broken (API mismatch → crashes)
It calls `calibrator.fit(hist_predictions_dict, hist_y, groups)` and `calibrator.predict(pred_dict, groups)`, but the real `AdaptiveTemporalECRCCalibrator` signature is `fit(y, pi, mu, r, *, groups)` / `predict(pi, mu, r, *, groups)`. The showcase simulator does not run, does not sweep the feedback gain κ, and uses only 10 steps.
- **Fix:** rewrite against the real API and sweep κ to *exhibit* the phase transition (spec in §3; the verified reference dynamics are in `src/civicsafe/theory/`).

### [D] 🟡 SAC loss still promoted in docs though disabled in code
`MATHEMATICS.md §10` sells the "propriety–sharpness tradeoff" as a contribution; the code correctly defaults it off. Keep it off; in the paper, present the sharpness penalty **only** as an ablation that *demonstrably hurts* calibration (that is a positive result — it validates using pure CRPS).

### [E] 🟡 MFFM diversity ≠ fairness
`MATHEMATICS.md §3` (JSD diversity regulariser) claims bias mitigation. Head-diversity is neither necessary nor sufficient for avoiding a protected-attribute proxy. Keep it as a representation-regulariser at most; do not claim it as a fairness guarantee.

### [F] 🟡 Stale docstring
`conformal.py:709` still shows the old flipped formula `alpha + gamma*(err - alpha)` while the code (`:885`) is correct. Fix the docstring to match.

### [G] 🟢 Trivial test failure
`tests/test_data.py:151` expects column `tract_id`; the ACS fallback emits `spatial_unit`. Update the assertion (not a real bug).

## 3. The upgrade delivered by this audit: the Feedback Amplification Law

The one genuinely novel, verified result is now **in the repo as tested code**:
- `src/civicsafe/theory/feedback_law.py` — the AOBF model, the universal amplification law, the runaway-discrimination corollary, and the passive/active identification duality.
- `tests/test_feedback_law.py` — **6 tests, all passing**, each verifying a theorem against the actual iterated feedback dynamics.

**The results (verified to ≥6 significant figures):**
1. **Universal Amplification Law:** at the feedback fixed point, `μ_s/μ_r = (λ_s/λ_r)^{1/(1−κ)}` with `κ = policy_elasticity × detection_elasticity`. Pole at `κ*=1`. Coordinate-free (holds for tanh policy + exponential detection, not just power laws).
2. **Runaway-Discrimination Corollary:** equal-truth groups diverge as `b^{1/(1−κ)}` — Ensign et al. (2018) made an exact law.
3. **Passive/Active Duality:** the "confidently wrong" state is invisible to passive observation (biased and honest worlds are observationally identical), but κ is **point-identified** by a difference-in-differences on log recorded rates after an exogenous detection shock — i.e. a staggered **ShotSpotter/patrol-policy natural experiment**. Verified: κ recovered to ±0.02.

**Why this is the paper:** it converts the project's thesis from "another calibrated crime forecaster" (incremental, scooped) into "a law governing when algorithmic allocation can be trusted at all, with an impossibility result and a field-identifiable parameter" (general, novel, cross-domain). It subsumes the fairness story (the Corollary), the ethics story (the impossibility), and the experimental story (the DiD) under one equation.
