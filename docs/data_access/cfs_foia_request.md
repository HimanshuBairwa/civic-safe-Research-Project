# 911 Calls-for-Service — Public-Records / FOIA Request (DRAFT)

*Use this ONLY if a city does not already publish calls-for-service (CFS) on its
open-data portal. Check the portal first (Chicago, NYC, Seattle, New Orleans,
Dallas, Detroit publish CFS with no request). Fill every [bracket]; adjust the
statute name to the city's state; delete this note before sending.*

---

**To:** [City] Police Department — Records / Public Information Office
(or the city's FOIA / public-records officer)
**Via:** [email / online portal / mail — use the address on the department's
"public records request" page]
**Date:** [date]
**Re:** Public Records Request — Calls-for-Service data, [YEAR RANGE]

To the Records Custodian,

Under **[STATE public-records law — e.g., the Illinois Freedom of Information Act,
5 ILCS 140 / the New York Freedom of Information Law (FOIL), Public Officers Law
§§84–90 / your state equivalent]**, I request records of **police calls for
service (CFS)** for **[CITY]** covering **[START DATE]–[END DATE]**, in a
**machine-readable format (CSV or equivalent)**, with the following fields per
call **where available**:

- date and time the call was received;
- call type / nature code and priority;
- **geographic unit** — police beat, precinct, or census tract (a
  **neighborhood-level identifier is sufficient; no street address is needed**);
- disposition, and whether an incident report was generated.

**Scope and privacy.** I am requesting **aggregate / de-identified call-level
records for academic research** on crime-measurement error. I do **not** seek
caller names, addresses, phone numbers, or any personally identifying
information — **please redact those fields**. A neighborhood/beat/precinct
geographic label is all that is required.

**Fee waiver.** I request a **fee waiver** as this is non-commercial research in
the public interest. If fees are unavoidable, please contact me before incurring
any charge exceeding **$[25]**.

**Format and partial release.** Electronic delivery (email or a download link) is
preferred. If any portion is exempt, please **release the remainder** and cite
the specific exemption for anything withheld.

Please acknowledge receipt and advise of the expected response date under the
applicable statute. Thank you for your assistance.

Sincerely,
[Your name]
[Affiliation / "Independent researcher"]
[Email] · [Phone optional] · [Mailing address if required]

---

### Before you send
- [ ] Confirm the city truly doesn't publish CFS (portal + a web search).
- [ ] Put the correct **state statute** name in the first paragraph.
- [ ] Find the department's official records-request channel (portal vs email).
- [ ] Keep the geographic ask at **beat/precinct/tract** — asking for exact
      addresses triggers privacy review and delays/denials.
